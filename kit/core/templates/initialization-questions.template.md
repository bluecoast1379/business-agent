# 初始化问题清单(INITIALIZATION_QUESTIONS)

> 各命令访谈中回答不了的问题集中登记在这里,回答后回填到对应产物文档。升级 kit 时本文件不被覆盖(preserveOnUpgrade)。
> 规则:问题与回答都只登记「键名与用途」级信息,严禁密钥值 / 内部域名端口 / 真实客户数据。

## 使用方式

1. 命令执行中遇到没有答案的事项 → 追加一行,标注来源命令与需要谁回答。
2. 拿到答案 → 勾选并把结论回填到对应产物文档(如 00-discovery.md),在「回填位置」记下章节。
3. 定期(建议每周)由 Owner 过一遍未答问题,催办或降级为「暂不需要」。
