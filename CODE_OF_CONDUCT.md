# 行为规范

## 我们的承诺

作为项目的成员、贡献者与维护者,我们承诺让每个人参与本项目的体验不受骚扰,无论其年龄、体型、可见或不可见的残障、民族、性别认同与表达、经验水平、教育背景、社会经济地位、国籍、外貌、种族、宗教或性取向如何。

## 我们的标准

有助于营造积极环境的行为包括:

- 对他人保持同理心与善意;
- 尊重不同的观点、经验与技术选择;
- 给出并坦然接受建设性反馈;
- 对自己的失误负责、向受影响者致歉,并从中改进;
- 关注对社区整体最有利的结果,而不只是个人立场。

不可接受的行为包括:

- 使用带性暗示的语言或图像,以及任何形式的性骚扰;
- 挑衅、侮辱或贬损性评论,人身攻击或政治攻击;
- 公开或私下的骚扰;
- 未经明确许可发布他人的私人信息(如住址、电子邮箱);
- 其他在专业场合中可被合理认定为不当的行为。

## 适用范围

本规范适用于所有项目空间(仓库、Issue、PR、讨论区),以及个人以项目代表身份出现的公开场合。

## 执行

违规行为可通过仓库维护者渠道(GitHub 私信或 Security 私密报告入口)举报。维护者有义务对举报保密,将及时、公正地评估并采取与情节相称的措施——从私下提醒、警告,到临时或永久禁止参与项目。

维护者若不善意执行本规范,也应承担相应责任。

## 归属

本规范参考了社区通行的贡献者公约精神撰写。
